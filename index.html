how it differs from a McKinsey engagement

Same frameworks · half the cost ceiling · cryptographic provenance.

  • Every claim is hash-chained. SHA-256 prevHash on every research entry. If a finding gets edited later, the chain breaks. Provenance built in.
  • Every deliverable is Ed25519-signed. Against your Konomi keypair. Client verifies "this exact analysis came from you at this exact time, untampered."
  • Adversarial verification on every claim. 3-vote pattern. Refuted claims get listed openly · same honesty pattern that won audiences for nhs-reinjection.
  • MIT open source. Fork it. Self-host. Brand it your firm's name. Same engine.
  • Sovereign. Single HTML file. Works offline. No backend. No leaking client data to a SaaS.
